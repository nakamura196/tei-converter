作成・編集・公開でたどる TEI/XML のワークフロー

1

作成する

📄

DOCX → TEI 変換

.docx ファイルを TEI Garage API で TEI/XML に変換します。
🔎

TEI Scanner

画像フォルダを Apple Vision で OCR し、facsimile zone 付きの TEI/XML を生成する macOS アプリ。

GitHub ↗ App Store ↗
2

編集する

✏️

TEI/IIIF エディタ ↗

TEI/XML と IIIF 画像を並べて翻刻・タグ付けを行う外部エディタです。別タブで開きます。

この工程で広く使われている定番ツール

Oxygen XML Editor ↗

TEI 編集の事実上の標準とされる商用 XML エディタ。TEI 用フレームワークを内蔵。

Scholarly XML (VS Code) ↗

VS Code で TEI/XML を編集するための拡張機能。RELAX NG による検証と入力補完。

LEAF-Writer ↗

ブラウザ上で動作するオープンソースの TEI/XML エディタ。
3

公開する

📝

TEI/XML ビューワ

TEI/XML ファイルをアップロードして構文ハイライトとプレビューで確認できます。
📚

TEI ギャラリー

XML 文書と XSL スタイルシートを左右に並べ、組み合わせて全画面の公開ビューを開きます。

この工程で広く使われている定番ツール

TEI Publisher ↗

TEI 文書をデジタル版として公開するためのプラットフォーム。

CETEIcean ↗

TEI/XML をブラウザでそのまま表示する JavaScript ライブラリ。本サイトのプレビューでも使用。
